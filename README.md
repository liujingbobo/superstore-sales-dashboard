# Superstore Sales & Profit Dashboard

This project analyzes the Superstore dataset using Python, Pandas, Matplotlib, Seaborn, and Tableau.

## Project Overview

The goal of this project is to explore:
- Sales performance
- Profitability by category
- Regional trends
- Discount impact on profit
- Top-performing products

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- Tableau Public

## Tableau Dashboard

[View Interactive Dashboard](https://public.tableau.com/views/SuperstoreSalesProfitDashboard_17792104892810/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

## Key Insights

- Technology generated the highest profit among all categories.
- Higher discounts were generally associated with lower profits.
- The West region achieved the strongest sales and profit performance.
- A small number of products contributed disproportionately high sales.
